pub unsafe fn _mm_test_mix_ones_zeros(a: __m128i, mask: __m128i) -> i32
This is supported on x86-64 and target feature sse4.1 only.

Tests whether the specified bits in a 128-bit integer vector are neither all zeros nor all ones.

Arguments:

  • a - A 128-bit integer vector containing the bits to be tested.
  • mask - A 128-bit integer vector selecting which bits to test in operand a.

Returns:

  • 1 - if the specified bits are neither all zeros nor all ones,
  • 0 - otherwise.
